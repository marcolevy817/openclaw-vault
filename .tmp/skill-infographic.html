🧩 What is an OpenClaw Skill?

A self-contained module that gives your AI agent a new capability

📦

Self-Contained Package

A skill is a folder with a SKILL.md at its root. Everything the AI needs to perform a task lives inside — instructions, scripts, references.

🧠

AI-Readable Instructions

The agent reads SKILL.md before acting. It describes what to do, how to do it, what tools to call, and edge cases to handle.

🔌

Pluggable & Shareable

Skills live in ~/.openclaw/skills/ or your workspace. Install from ClawHub, share with others, or write your own from scratch.

🎯

Triggered by Context

The agent picks the right skill based on your request. Each skill has a description that helps match it to the right situation automatically.

📁 Skill File Structure

my-skill/Root folder — name is the skill ID
  SKILL.md⭐ Core instructions — the agent reads this before acting
  references/Docs, API references, data files the agent can consult
  scripts/Executable helpers the agent can run during a task
  assets/Images, templates, or other supporting files
